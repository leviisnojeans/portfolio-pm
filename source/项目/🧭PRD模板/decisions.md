## 决策 #1（2026-MM-DD）

- **决策**：礼物面板采用分 Tab 结构（推荐/节日/全部/我的），不用长滚动

- **影响原型版本**：v1.0 起生效
- **干系人确认**：设计 ✅ / 客户端 ✅
- **可推翻条件**：灰度发现 Tab 切换损耗 > 8%

## 决策 #2（2026-MM-DD）

- **决策**：试水档定为 ¥1 / ¥3 / ¥6 三档；¥1 "拍拍" 不参与主播分成，由平台 100% 补贴
- **背景**：
  - 总监例会："不要做放水型补贴"（[`inputs/business-asks.md`](inputs/business-asks.md)）
  - 商业化测算：¥1 参与分成会让主播 ARPPU 报表显著虚高
  - 子话题 02 判断：¥1 不分成可以解释为"用户表达意图，主播不必有压力回应"
- **影响 PRD 章节**：4.2、9（风险表）
- **干系人确认**：商业化 ✅ / 法务 🟡（合规性待复核）
- **可推翻条件**：法务认定 ¥1 不分成存在合规问题
