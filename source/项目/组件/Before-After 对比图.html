Before / After Hover 对比图

纯 CSS 实现，默认显示「改版后」，hover 显示「改版前」。
将鼠标悬停在下图查看效果 ↓

After Before
使用说明： 将上面 /* 组件 CSS */ 复制到你的 <style> 中， 将 HTML 结构复制到页面合适位置，替换图片 src 即可。
HTML 模板（直接复制）
<div class="comp">
  <img class="after"  src="改版后.png" alt="改版后">
  <img class="before" src="改版前.png" alt="改版前">
  <span class="comp-badge after">After</span>
  <span class="comp-badge before">Before</span>
</div>
可调参数
aspect-ratio: 16 / 9容器宽高比，可改为 4/3、1/1 等
object-fit: contain改为 cover 则图片裁剪铺满
0.35s过渡速度，调大更慢、调小更快
background: #1a1a1a图片未加载时的占位背景色
行为说明
状态显示标签
默认改版后（After）青色「After」
Hover改版前（Before）淡入琥珀色「Before」淡入
移动端始终显示改版后不受影响
